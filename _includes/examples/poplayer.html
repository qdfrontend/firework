弹出层详细文档

示例

具有多个弹出效果，支持回调，事件监听。

使用前一定要初始化

目前功能比较单一，不够完善，使用前一定先要初始化一下。

显示与隐藏

找到对应的元素进行初始化，并调用显示或隐藏方法。

{% highlight html %}
//控件初始化 $("#poplayer-demo").poplayer(); $("#poplayer-show").click(function(){ //调用显示方法，并传入defaultStyle显示方式 $("#poplayer-demo").poplayer("show"); }); $("#poplayer-hide").click(function(){ //隐藏方式在显示的时候就设置好了，不需要传入defaultStyle值。 //调用隐藏方法。 $("#poplayer-demo").poplayer("hide"); }); {% endhighlight %}

参数

设置弹出方式

通过设置defaultStyle选项值可改变弹出方式。或初始化时指定默认方式

名称 类型 默认值 描述
defaultStyle 字符串'string' "fadinscale"（默认是它）, "slideinright", "slidbottom", "newspaper", "fall", "sidefall", "stickup", "fliphorizontal", "flipvertical", "sign", "superscaled", "justme", "slit", "rotatebottom", "rotateleft", "blur", "letin", "makeway" "sliptop" defaultStyle控制显示与隐藏的动画方式。

方法

.poplayer("show",[defaultStyle])

显示对话框。接受可选参数 defaultStyle。

{% highlight js %} $("#el").poplayer("show",[defaultStyle]); {% endhighlight %}

.poplayer("hide")

显示对话框。接受可选参数。

{% highlight js %} $("#el").poplayer("hide"); {% endhighlight %}

事件

Bootstrap 的模态框类提供了一些事件用于监听并执行你自己的代码。

事件类型 描述
poplayerhideanimateend 显示完成。
poplayershowanimateend 隐藏完成。
{% highlight js %} // 监控一个事件要使用如下规则 // 控件名称+事件名字 // 注意 要全部小写 例如poplayer +hideAnimateEnd == poplayerhideanimateend $("#poplayer-demo2").on("poplayershowanimateend",function(){ // do something... }); $("#poplayer-demo2").on("poplayerhideanimateend",function(){ // do something... }); {% endhighlight %}
