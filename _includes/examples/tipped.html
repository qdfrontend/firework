自定义Tip组件详细文档

示例

以下是各种不同皮肤的样式 

 <div class='demonstrations'>
   <div id='demo_skins_dark' title="This is the default skin"><!-- [] --></div>
   <div id='demo_skins_black' title="The same skin without a radius"><!-- [] --></div>
   <div id='demo_skins_light' title="A light variant"><!-- [] --></div>
   <div id='demo_skins_white' title="White is probably the most popular skin"><!-- [] --></div>
   <div id='demo_skins_gray' title="Gray works with anything"><!-- [] --></div>
   <div id='demo_skins_yellow' title="The classic yellow tooltip"><!-- [] --></div>
   <div id='demo_skins_blue'><!-- [] --></div>
   <div id='demo_skins_red'><!-- [] --></div>
   <div id='demo_skins_green'><!-- [] --></div>
   <div id='demo_skins_tiny' class='rightmost'><!-- [] --></div>
 </div>

<script type="text/javascript">
 jQuery(document).ready(function($) {
   Tipped.create("#demo_skins_dark");
   Tipped.create("#demo_skins_black", { skin: 'black' });
   Tipped.create("#demo_skins_light", { skin: 'light' });

   Tipped.create("#demo_skins_white", { skin: 'white' });
   Tipped.create("#demo_skins_yellow", { skin: 'yellow' });
   Tipped.create(".demo_skins_yellowB", "This is a new TIP",{ skin: 'yellow' });
   Tipped.create("#demo_skins_gray", { skin: 'gray' });
   Tipped.create("#demo_skins_yellowA", { skin: 'yellow' });
   Tipped.create("#demo_skins_blue", "Skins are optimized to look good on any background", { skin: 'blue' });
   Tipped.create("[name*=demo_skins_redA]", "This is a new TIP", { skin: 'red' });
   Tipped.create("#demo_skins_red", "This is a new TIP", { skin: 'red' });
   Tipped.create("#demo_skins_green", "A nice green skin", { skin: 'green' });

   Tipped.create("#demo_skins_tiny", "Small black tooltips are always useful", { skin: 'tiny' });
 });
</script>

参数

设置弹出tip的样式

名称 类型 默认值 描述
id 字符串'string' skin 设置不同皮肤的tip     "dark" "black", "light", "white", "gray", "yellow", "blue",     "red", "green", "tiny" id控制tip的颜色

方法

你都可以通过设置不同的id,class，name甚至其他自定义属性来进行控制,如:

①使用设置id的方式：

1.在html中创建一个元素，例如你将给一个DIV的tip改成黄色，你用设置ID的方式，title的内容就是tip里的内容：
"<div id='demo_skins_yellow' title="This is a demo"><!-- [] --></div>"
2.然后绑定弹出tip事件，传三个参数：需要绑定的元素，tip里的内容，和弹框的颜色（不设置默认为dark）： 
<script type='text/javascript'>
    jQuery(document).ready(function($) {
  Tipped.create("#demo_skins_yellowA",{ skin: 'yellow' });
  });
</script>

②使用设置class的方式：

1.在html中创建一个元素，例如你将给一个DIV的tip改成黄色，你也可以用设置class的方式：
"<div class='demo_skins_yellow' title="This is a demo"><!-- [] --></div>"
2.然后绑定弹出tip事件： 
<script type='text/javascript'>
    jQuery(document).ready(function($) {
  Tipped.create(".demo_skins_yellowB","This is a new TIP" { skin: 'yellow' });//还需要说的一点如果传一个新的title内容的参数如："This is a new TIP"，那么这个参数会替换掉原来的title的内容
  });
</script>

③使用设置自定义属性的方式：

1.在html中创建一个元素，例如你将给一个DIV的tip改成黄色，你也可以用设置自定义属性的方式，例如使用name：
"<div name='demo_skins_redA' title="This is a demo"><!-- [] --></div>"
2.然后绑定弹出tip事件： 
<script type='text/javascript'>
    jQuery(document).ready(function($) {
  Tipped.create("[name=demo_skins_redA]","This is a new TIP" { skin: 'red' });//还需要说的一点如果传一个新的title内容的参数如："This is a new TIP"，那么这个参数会替换掉原来的title的内容
  });
</script>

至此你应该可以发现，你可以使用jq任何选择器来绑定添加Tipped组件/font>
