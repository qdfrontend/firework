ui.plupload上传文件详细文档

ui.plupload 上传附件

Plupload是有TinyMCE的开发者开发的，为您的内容管理系统或是类似上传程序提供一个高度可用的上传插件。Plupload 目前分为一个核心API 和一个jQuery上传队列部件，这样使你可以直接使用或是自己定制。

注意：

  • 分块:chrome和firefox 4+ 支持。
  • 拖拽适用于firefox和webkit内核的浏览器，windows版的safari还存在一些问题待解决。
  • 图片缩放仅在firefox3.5+和chrome上支持，safari/opera 不支持直接数据访问选定的文件。
  • 目前所有的浏览器不支持文件类型过滤。但是，我们填写HTML5的接受文件类型过滤属性，一旦有支持它会工作。
  • 多文件上传仅支持gecko和webkit内核的浏览器。

使用方法

下载
plupload：http://www.plupload.com/examples/uiwww.plupload.com/examples/ui
jquery-ui：http://jqueryui.com/
页面引入： jquery-ui.css 、jquery.ui.plupload.css、 jquery-1.8.3.js、 jquery-ui.js、 plupload.full.min.js、 jquery.ui.plupload.js

页面添加

  1. <div id="uploader">
  2.     <p>Your browser do not have Flash, Silverlight or HTML5 support.</p>
  3. </div>

JS添加1

  1. $("#uploader").plupload({
  2.         runtimes : 'html5,flash,silverlight,html4',
  3.         url : "uploads", 
  4.         max_file_size : '2mb',//文件上传最大限制
  5.         chunk_size: '1mb',//上传分块每块的大小
  6.         resize : {
  7.             width : 200,
  8.             height : 200,
  9.             quality : 90,
  10.             crop: true 
  11.         },//缩略图
  12.         filters : [
  13.             {title : "Image files", extensions : "jpg,gif,png"},
  14.             {title : "Zip files", extensions : "zip,avi"}
  15.         ],//上传文件时限制的上传文件类型
  16.         rename: true,//上传之前可重命名
  17.         sortable: true,
  18.         dragdrop: true,//可拖拽
  19.         views: {
  20.             list: true,
  21.             thumbs: true, 
  22.             active: 'thumbs'
  23.         },
  24.         flash_swf_url : 'js/Moxie.swf',//flash文件地址
  25.         silverlight_xap_url : 'js/Moxie.xap'//silverlight文件地址
  26.     });
